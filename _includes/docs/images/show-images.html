Show Images

Command

rumm show images

Options

none

Arguments

none

Description

Shows a list of all of the images available to the active Rackspace account, both Rackspace base images and user-created snapshot images.

Examples

rumm show images
  3a23d0a-de1b-4eb7-994f-5f0de9d8d7a0 Fedora 19 (Schrodinger's Cat)
  ccaf99bc-472a-46ea-a125-d3ecfca66695 FreeBSD 9.1
  16e6c0ae-f881-4180-95b0-3450fe3f8e96 Red Hat Enterprise Linux 6.4
  8c271f6e-f325-4963-b9d5-61f984ebcc20 Red Hat Enterprise Linux 6.3
  fca369d4-e4c0-4659-9b43-d2e5702d5df1 Red Hat Enterprise Linux 6.1

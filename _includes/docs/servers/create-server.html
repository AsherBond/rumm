Create Server

Command

rumm create server [--name STRING] [--image-id STRING] [--flavor-id STRING]

Options

-n, --name STRING               # Name to give the new server
-i, --image-id STRING           # Use this VM image
-f, --flavor-id STRING          # Create with the specified flavor

Arguments

none

Description

Creates a new server. If a name is not provided, one will be generated automatically. If you want to base the server on an existing image, you will need to specify the image id. Similarly, a flavor id can be specified. The defualt is a 512MB sever running Ubuntu 12.04 LTS. This command will take a few minutes.

Examples

rumm create server
  --> bootstrapping server smoggy-shigellosis

rumm create server --name rumm-example-server
  --> bootstrapping server rumm-example-server

rumm create server --image-id 9922a7c7-5a42-4a56-bc6a-93f857ae2346 --flavor-id 3
  --> bootstrapping server happy-holiday

Create Server

Command

rumm create server [--name STRING] [--image-id STRING] [--flavor-id STRING] [--private-key-pathSTRING] [--public-key-pathSTRING]

Options

    --name STRING                      # Name to give the new server
    --image-id STRING                  # Use this VM image
    --flavor-id STRING                 # Create with the specified flavor
    --private-key-path STRING          # Path to your private ssh key
    --public-key-path STRING           # Path to your public ssh key

Arguments

none

Description

Creates a new server. If a name is not provided, one will be generated automatically. If you want to base the server on an existing image, you will need to specify the image id. Similarly, a flavor id can be specified. The default is a 512MB server running Ubuntu 12.04 LTS. This command will take a few minutes.

rumm assumes that your ssh keys are at ~/.ssh/id_rsa.pub and ~/.ssh/id_rsa. If that is not the case, you will need to specify the path to your keys with the public-key-path and private-key-path options.

Examples

rumm create server
  --> bootstrapping server smoggy-shigellosis
      done.

rumm create server --name rumm-example-server
  --> bootstrapping server rumm-example-server
      done.

rumm create server --image-id 9922a7c7-5a42-4a56-bc6a-93f857ae2346 --flavor-id 3
  --> bootstrapping server sesquipedalian-stereotype
      done.

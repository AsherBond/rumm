Show Server

Command

rumm show server ID

Options

none

Arguments

ID: STRING          # Name of the server to show

Description

Shows the attributes of the sever with the name specified by ID.

Examples

rumm show server sorrel-shantung
  sorrel-shantung: f89c4f57-f66b-4a14-aae2-e3dedf7f59b2 (ACTIVE)
    Ubuntu 13.04 (Raring Ringtail)  512MB Standard Instance
    created: 2013-07-01T15:38:15Z
    ipv4: 166.78.111.48
    ipv6: 2001:4801:7817:72:2dc2:cd85:ff10:a64a

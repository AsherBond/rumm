Show Servers

Command

rumm show servers

Options

none

Arguments

none

Description

Shows a list of all of the servers contained in the current user's Rackspace account.

Examples

rumm show servers
  sorrel-shantung -> id: f89c4f57-f66b-4a14-aae2-e3dedf7f59b2, status: ACTIVE, ipv4: 166.78.111.48
  smoggy-desktop -> id: f89c4f57-f66b-4a14-aae2-e3dedf7f59b2, status: ACTIVE, ipv4: 166.78.111.48

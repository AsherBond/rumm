Create File

Command

rumm create file ID in container CONTAINER_ID

Options

none

Arguments

ID: STRING                       # ID of the file to create
CONTAINER_ID: STRING          # Name of the container to place the file in

Description

Creates a new file in the specified container.

Examples

rumm create container
  created container: dragging-ceratopogon

rumm create file new-file in container dragging-ceratopogon

Show Load Balancer

Command

rumm show loadbalancer ID

Options

none

Arguments

ID: STRING          # Name of the load balancer to show

Description

Shows the specified load balancer.

Examples

rumm show loadbalancer located-bruce
  located-bruce: 154461 (ACTIVE)
    RANDOM
    created: {"time"=>"2013-07-15T19:36:00Z"}
    protocol: HTTP

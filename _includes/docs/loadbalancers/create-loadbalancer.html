Create Load Balancer

Command

rumm create loadbalancer [--name STRING] [--port INTEGER] [--protocol STRING] [--virtual-ips STRING]

Options

    , --name                 # The name of the load balancer
    , --port                 # The port of the load balancer
    , --protocol             # The protocol to be used
    , --virtual-ips          # A list of virtual IPs to be used
                             #  arguments:
    , --node                 # A node for the load balaner
                             #  arguments:

Arguments

none

Description

Creates a new loadbalancer. If the load balancer is not named, a name will be generated automatically. The default port is 80, and hte default protocol is HTTP.

The available protocols are HTTP, and ??

The available virtual IP types are PUBLIC and SERVICENET.

Example

rumm create loadbalancer --node 10.0.0.1:80

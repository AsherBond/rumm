Show DB Instance

Command

rumm show dbinstance ID

Options

none

Arguments

ID: STRING          # Name of the DB instance to show

Description

Shows the specified DB instance.

Example

rumm show dbinstance
  drum-shaped-inerrancy -> id: 1c2c657b-f89d-4623-9239-fc944d1d60ec, hostname: 4c71fbb7ca2f3aa8bb387befbfda118f6cea26c0.rackspaceclouddb.com, status: ACTIVE

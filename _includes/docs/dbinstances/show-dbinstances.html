Show DB Instances

Command

rumm show dbinstances

Options

none

Arguments

none

Description

Show a list of DB instances contained in the current Rackspace account.

Example

rumm show dbinstances
  departmental-ichthyologist -> id: 08306258-d60a-41c3-b926-371681e16a3f, status: BUILD
  drum-shaped-inerrancy -> id: 1c2c657b-f89d-4623-9239-fc944d1d60ec, status: ACTIVE

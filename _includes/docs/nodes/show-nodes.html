Show Nodes

Command

show nodes on loadbalancer LOADBALANCER_ID

Options

none

Arguments

LOADBALANCER_ID          # The name of the load balancer that owns the nodes to be listed

Description

Shows all the nodes on the named load balancer.

Examples

rumm show nodes on loadbalancer loamy-bradypodidae
  id: 390579
  id: 390581
  id: 390583

Show Users

Command

rumm show users on dbinstance DBINSTANCE_ID

Options

none

Arguments

DBINSTANCE_ID: STRING          # Name of the DB instance on which the users can be found

Description

Shows all of the users in the specified DB instance. The databases to which the users have access are listed after the username.

Examples

rumm show users on dbinstance departmental-ichthyologist
  me, [{"name"=>"different-boycott"}, {"name"=>"disentangled-behmen"}]
  you, [{"name"=>"different-boycott"}]

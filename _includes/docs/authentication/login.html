Login

Command

rumm login

Options

none

Arguments

none

Description

Logs the user into a Rackspace account. You will be prompted to enter your Rackspace username and password. The username and your Rackspace API key (not your password) will be stored in ~/.netrc. You only need to do this once. The Logout command can be used to remove your credentials from ~/.netrc.

Rumm uses the credentials for the cloud portal (https://mycloud.rackspace.com/), not the managed portal (https://my.rackspace.com/portal/home).

Examples

rumm login
  username: rstevenson
  password: *********
  logged in as rstevenson, credentials written to ~/.netrc
